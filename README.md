This project is a **job portal** where users can search, filter, and manage job offers.  
It includes advanced form validation, real-time feedback, CRUD operations, a favorites system, and user profile management.  
The project uses **Vanilla JavaScript only** (no frameworks or libraries) and is designed to be fully responsive.

## ğŸš€ Features

### 1. Form Validation
- **User Profile**: Validates name, email, and skills.  
- **Job Offer Form**: Validates required fields (title, company, description).  
- **Real-time Feedback**: Displays success or error messages during input.

### 2. Advanced Search & Filtering
- **Text Search**: Search through titles, companies, and descriptions.  
- **Multiple Filters**: Filter by skills, contract type, and location.  
- **Combined Filtering**: Combine search and filters for refined results.

### 3. Complete CRUD Operations
- **Add**: Modal form for adding new job offers.  
- **Edit**: Modify existing job offers.  
- **Delete**: Remove offers with confirmation and UI updates.  
- **Display**: Responsive job cards layout.

### 4. Favorites System
- **Add/Remove**: Mark offers as favorites.  
- **Favorites Tab**: View all favorites in a dedicated section.  
- **Persistence**: Favorites are saved automatically in `localStorage`.

### 5. Profile Management
- **Skills**: Add or remove skills.  
- **Preferences**: Set job type and preferred location.  
- **Persistence**: Profile data is automatically saved.

### 6. Event Management
- **Offer Interactions**: Clicks, favorites, and edits.  
- **Tab Navigation**: Switch between All Offers, Favorites, and My Offers.  
- **Form Events**: Submission, validation, and cancellation handling.
